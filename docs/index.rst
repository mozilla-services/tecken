=====================
Mozilla Symbol Server
=====================


Mozilla Symbol Server, codename "tecken", is a web service for handling
all things symbols for Mozilla platform. In particular three major features:

1. Symbolication

2. Symbol Download

3. Symbol Upload (under development)
