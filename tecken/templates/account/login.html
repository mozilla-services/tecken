What is this?

The Telemetry Analysis Service gives direct access to Mozilla Telemetry data:

  • Perform interactive analyses with Apache Spark and Jupyter.
  • Run custom map-reduce analyses with the Telemetry MapReduce framework.
  • Schedule these analyses so they run on a regular basis, publishing output to Amason S3.
